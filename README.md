# TweetsAnalysisLDA

Requirements:

1. [x] Git
2. [x] Python 3.9
3. [x] Pycharm IDE


How to run?:
1. run this command `pip install -r requirements.txt`
2. run this command for additional package too `pip install git+https://github.com/JustAnotherArchivist/snscrape.git` (MUST)
3. Go to bin folder
4. Run main.py
